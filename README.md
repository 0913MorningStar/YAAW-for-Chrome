#YAAW for Chrome

简单的把YAAW打包发布到Chrome上,安装之后会出现YAAW的图标.

#Usage


点击YAAW图标便会出现YAAW的管理界面,设置好 JSON-RPC Path即可.

之后任意链接都可以右键导出到YAAW进行下载.


#Install
作为chrome扩展安装即可.

Web Store link : 

https://chrome.google.com/webstore/detail/yaaw-for-chrome/dennnbdlpgjgbcjfgaohdahloollfgoc
